到软件的根目录下 执行以下脚本
sudo ./set_ptenv.sh sudo ./set_qtenv.sh

**一定按步骤走 **
安装在默认的/opt/pt 如果因为安装在家目录而导致闪退 就重新安装在默认路径下

一些依赖库
sudo apt install libqt5scripttools5
sudo apt-get install libqt5multimedia5-plugins
